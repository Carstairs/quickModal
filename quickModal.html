QuickModal: Lightweight Modal Dialog

Demo

General Info

quickModal is a jQuery plug in that creates a modal dialog from a div on the page, using the trigger's href attribute to set which div to display.

Features

  • Multiple instances on a single page
  • Customizable via plug in options and css.
  • Lightweight (1.8kb minimized)
  • Clicking out of the modal ara closes modal (i.e., close button optional)

How To Set Up & Use QuickModal

  1. Add the plug-in script to the page, after you've added jQuery. 
    <script src="/path-to-scripts/jquery.quickModal.min.js" 
 type="text/javascript"></script>
  2. Add the modal's css to your css file, or to the page 
    <link href="/path-to-styles/quickModal.css" rel="stylesheet" />
  3. Call the plug in, using your selector's id or class 
    $('.modalTrigger').quickModal();
  4. Set any options you want to customize 
    $('.modalTrigger').quickModal({ 
    overlayColor: "navy", 
    borderRadius: "0", 
    modalWidth: "700" 
});
  5. When the user clicks on your selector, the modal will quickly fade in. Clicking on the element with the closeButton class, or anywhere outside of the modal, will close the modal.

Notes

  • The page behind the modal will not scroll while the modal is open.
  • The term "overlay" refers to the partial opacity background behind the modal. (You've probably already worked that out, but just in case.)

Options

The plug in has lots of settings you can tweak.

Option Default Value
closeButton".modal-close"
overlayOpacity0.5
overlayColor"#000"
overlayZIndex100
modalZIndex1000
modalWidth"500"
boxShadow"0px 0px 10px rgba(0,0,0,0.5)"
borderCss"1px solid #444"
borderRadius"4px"
modalBgColor"#fff"
headerBgColor"rgb(185, 185, 185)"
footerBgColor"rgb(219, 219, 219)"
